# Java Production Management Project

##  Description
Ce projet est une application Java destinée à la gestion de la production et des opérateurs.  
Il fournit deux modules principaux :
- **ProductionManager** : gère les opérations liées à la production (planification, suivi, rapports).
- **OperatorManager** : gère les informations et activités des opérateurs (assignations, disponibilité, performances).

L’objectif est de fournir une base solide pour un système de gestion de production, extensible et maintenable.


## Prérequis
- **Java 17** ou version supérieure
- **Git** pour le contrôle de version
- Un IDE (IntelliJ, Eclipse, VS Code) ou un terminal avec `javac` et `java`
